    1.递归 – 循环

    ​ 循环：你打开面前这扇门，看到屋里面还有一扇门。你走过去，发现手中的钥匙还可以打开它，你推开门，发现里面还有一扇门（若前面两扇门都一样，那么这扇门和前两扇门也一样；如果第二扇门比第一扇门小，那么这扇门也比第二扇门小，你继续打开这扇门，一直这样继续下去直到打开所有的门。但是，入口处的人始终等不到你回去告诉他答案。

    ​ 递归：你打开面前这扇门，看到屋里面还有一扇门。你走过去，发现手中的钥匙还可以打开它，你推开门，发现里面还有一扇门，你继续打开它。若干次之后，你打开面前的门后，发现只有一间屋子，没有门了。然后，你开始原路返回，每走回一间屋子，你数一次，走到入口的时候，你可以回答出你到底用这你把钥匙打开了几扇门。

    2.定义 - 举例

    (1) 递归 - 定义

    ​ 在高级语言中，调用自己和其他函数并没有本质的不同。我们把一个直接调用自己 或通过一系列的调用语句间接地调用自己的函数，称做递归函数

    (2) 在程序中，所谓的递归，就是函数自己直接或间接的调用自己。调用自己分两种：

    1. 直接调用自己
    2. 间接调用自己

    就递归而言最重要的就是跳出结构，因为跳出了才可以有结果.

    递归 - 举例
    直接调用
    // 直接调用
#include 

using namespace std;

void recursion()
{
   recursion(); /* 函数调用自身 */
}

int main()
{
   recursion();
}
    间接调用
    // 间接调用

#include 

using namespace std;

void Recursion_1(); // 函数声明
void Recursion_2();

void Recursion_1() {
    Recursion_2();  // 调用 第二个函数
}

void Recursion_2() {
    Recursion_1();  //调用 第一个函数
}

int main() {
    Recursion_1();
    return 0;
}

    例题

    1. N!

    题目:从键盘输入一个数，求出这个数的阶乘，即 n!
    N! – 循环版
    // N! -- 循环版

#include 
using namespace std;

int main() {
    int n, sum = 1;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        sum *= i;
    }
    cout << sum << endl;
    return 0;
}
    N! – 递归版
    // N! -- 递归版

#include 
using namespace std;

int factorial(int n) {
    if (n == 1) return 1;
    return n * factorial(n - 1);
}

int main() {
    int n;
    cin >> n;
    cout << factorial(n) << endl;
    return 0;
}

    2. 数学家莱昂纳多·斐波那契

    这里咱们叫他老斐

    老斐以兔子繁殖为例子给我们讲了一下什么 斐波那契数列

    一如果说兔子在出生两个月后，就有繁殖能力，一对兔子每个月能生出一对小兔子来。假设所有兔子都不会死去，能够一直干下去，那么一年以后可决繁殖多少对兔子呢?

    斐波那契 – 循环版
    // 斐波那契 -- 循环版

#include 
#define MAX_N 100   // 宏定义

using namespace std;

int main() {
    int Months[MAX_N] = {}; // 这里咱们以兔子举例 所有用月数表示
    Months[0] = 0;  // 它们还没相识
    Months[1] = 1;  // 它们刚谈上甜甜的恋爱！！！
    cout << Months[1]; // 输出一月
    for (int i = 2; i < 40; i++) {
        Months[i] = Months[i - 1] + Months[i - 2];
        cout << Months[i] << " ";
    }
    return 0;
}
    斐波那契 – 递归版
    // 斐波那契 -- 递归版

#include 

using namespace std;

int Fibonacci(int M) {
    // if (M < 2) return M == 0 ? 0 : 1; // 思考 是可有可无么
    return Fibonacci(M - 1) + Fibonacci(M - 2);
}

int main() {
    int Months = 5;
    cout << Fibonacci(Months) << " ";
    return 0;
}

    什么是汉诺塔

    约19世纪末，在欧州的商店中出售一种智力玩具，在一块铜板上有三根杆，最左边的杆上自上而下、由小到大顺序串着由64个圆盘构成的塔。目的是将最左边杆上的盘全部移到中间的杆上，条件是一次只能移动一个盘，且不允许大盘放在小盘的上面。

    这是一个著名的问题，几乎所有的教材上都有这个问题。由于条件是一次只能移动一个盘，且不允许大盘放在小盘上面，所以64个盘的移动次数是：18,446,744,073,709,551,615

    这是一个天文数字，若每一微秒可能计算(并不输出)一次移动，那么也需要几乎一百万年。我们仅能找出问题的解决方法并解决较小N值时的汉诺塔，但很难用计算机解决64层的汉诺塔。
